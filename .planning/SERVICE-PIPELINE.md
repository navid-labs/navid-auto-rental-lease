## 4. E2E 데모 시나리오 (투자자 프레젠테이션)

Phase 9 Success Criteria #5: "End-to-end demo flow works without errors"

### 시나리오 A: 고객이 차량을 찾고 렌탈 계약하기

```
1. [랜딩 페이지] 방문
   → 히어로 섹션, 퀵서치, 피처드 차량 확인

2. [차량 검색] "현대 아반떼" 검색
   → 필터: 브랜드=현대, 모델=아반떼, 가격=2000-3000만원
   → URL 상태 유지 확인 (북마크 가능)

3. [차량 상세] 아반떼 1.6 GDi 클릭
   → 사진 갤러리, 제원, 가격 정보 확인
   → 월납입금 계산기 사용 (보증금 10%, 36개월)
   → 렌탈 vs 리스 비교 확인

4. [계약 신청] "렌탈 신청" 클릭
   → 4단계 위저드 진행 (차량→조건→eKYC→리뷰)
   → Mock eKYC 본인인증 완료
   → 계약 제출

5. [마이페이지] 계약 상태 확인 ← Phase 8
   → "승인 대기" 상태 표시
   → 실시간 업데이트 확인

6. [관리자 승인] 관리자 계정 전환
   → 계약 승인 큐에서 해당 계약 승인

7. [마이페이지] 고객 계정으로 복귀 ← Phase 8
   → "승인됨" 상태 실시간 반영 확인
   → 계약서 PDF 다운로드
```

### 시나리오 B: 딜러가 차량을 등록하고 승인받기

```
1. [딜러 회원가입] → 역할 부여
2. [딜러 대시보드] → "차량 등록" 클릭
3. [번호판 조회] → 12가1234 입력 → 자동 정보 채움
4. [정보 입력] → 가격/색상/주행거리 설정
5. [사진 업로드] → 5장 업로드 + 순서 변경
6. [등록 완료] → "승인 대기" 상태 확인
7. [관리자 승인] → 관리자가 차량 승인
8. [딜러 확인] → 승인 상태 반영 확인
9. [공개 검색] → 승인된 차량이 검색 결과에 노출
```

### 시나리오 C: 관리자 운영 플로우

```
1. [관리자 대시보드] → 통계 확인 (차량수/계약수/사용자수) ← Phase 9
2. [차량 승인 큐] → 딜러 차량 3건 배치 승인
3. [자사 차량 등록] → 관리자 직접 차량 등록
4. [잔존가치 관리] → 현대 아반떼 2024 36개월 잔가율 수정
5. [계약 관리] → 대기 중 계약 승인/반려
6. [사용자 관리] → 신규 딜러 역할 부여 ← Phase 9
```
