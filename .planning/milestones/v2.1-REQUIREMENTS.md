# Requirements: Navid Auto -- v2.1 Visual Polish

**Defined:** 2026-03-23
**Core Value:** K Car 수준의 여백과 시각적 완성도를 전 페이지에 적용

## v2.1 Requirements

크롬 비주얼 점검 + K Car 원본 비교 기반. 전 페이지 여백/미적 완성도 개선.

### 홈페이지

- [x] **HOME-01**: 홈페이지 주요 섹션(히어로, 검색, 퀵링크, 추천차량, 프로모션, 파트너) 간 세로 여백을 80px 이상으로 확대
- [x] **HOME-02**: 추천차량 그리드를 4열에서 3열로 변경하여 카드 간 여유 확보
- [x] **HOME-03**: "어떤 차를 찾으세요?" 검색 섹션 내부 패딩 확대 (타이틀↔탭↔인풋 간격 40px+)
- [x] **HOME-04**: 프로모션 카드(홈서비스, 위클리특가, 렌트) 간 gap을 24px로 확대

### 검색 페이지

- [x] **SRCH-01**: 차량 카드 그리드 gap을 16px에서 24px로 확대
- [x] **SRCH-02**: 검색바→브레드크럼→필터/카드 콘텐츠 간 전환 여백 24px 이상으로 확대
- [x] **SRCH-03**: 차량 카드 내부 이미지→텍스트 전환 영역 패딩 확대 (12px → 16px+)
- [x] **SRCH-04**: 퀵필터 pill 영역 상하 여백 16px 추가

### 상세 페이지

- [x] **DETL-01**: 갤러리 탭↔네비게이션 바 사이에 브레드크럼과 적절한 여백(24px) 추가
- [x] **DETL-02**: 비슷한 차량 추천 그리드를 4열에서 3열로 변경
- [x] **DETL-03**: 상세 페이지 각 정보 섹션 카드 간 세로 간격을 32px로 통일

### 글로벌 공통

- [x] **GLBL-01**: 네비게이션 바 높이를 44px에서 52px로 확대
- [x] **GLBL-02**: 모든 페이지의 콘텐츠 시작 상단 여백을 24-32px로 통일
- [x] **GLBL-03**: 어드민 대시보드에 동일한 여백 규칙 일괄 적용
