    - /mypage/* redirects to /login if not authenticated

    - Authenticated users visiting /login or /signup are redirected to role-appropriate dashboard

         ```
         '/dealer': ['DEALER', 'ADMIN']
         '/mypage': ['CUSTOMER', 'DEALER', 'ADMIN']
         ```
