    4. Create src/features/vehicles/actions/restore-vehicle.ts ('use server'):
       - Admin only — sets status back to AVAILABLE
       - Creates VehicleStatusLog entry
    5. Create src/features/vehicles/actions/update-status.ts ('use server'):
       - Takes vehicleId, newStatus, optional note
