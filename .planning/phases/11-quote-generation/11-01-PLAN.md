1. Create `src/features/inventory/types/quote.ts`:
   - Define `InventoryVehicleForQuote` type with fields needed from Phase 10's InventoryItem: `{ id: string, vehicleName: string, vehiclePrice: number, brand: string, isImport: boolean, vehicleCategory: VehicleCategory, fuelType: FuelType, promotionRate?: number, subsidyAmount?: number, options?: string, exteriorColor?: string, year?: number }`. Import VehicleCategory and FuelType from `@/lib/finance`.
   - Define `QuoteParams` type: `{ leasePeriodMonths: number, residualMethod: ResidualMethod, residualRate: number, depositRate: number, advancePayment: number, creditGroup: 1 | 2 | 3 }`. Import ResidualMethod from `@/lib/finance`.
   - Define `VehicleQuoteResult` type: `{ vehicle: InventoryVehicleForQuote, leaseResult: QuoteResult, rentalEstimate: { monthlyPayment: number, totalPayment: number }, effectivePrice: number }`. The `effectivePrice` = vehiclePrice - subsidyAmount - (vehiclePrice * promotionRate). The `rentalEstimate` uses `estimateMonthlyRental` from `@/lib/finance` for a simple rental comparison.
   - Define `QuoteGenerationResult` type: `{ params: QuoteParams, vehicles: VehicleQuoteResult[], generatedAt: Date }`.

2. Create `src/features/inventory/schemas/quote-schema.ts`:
   - Zod schema `quoteParamsSchema` validating QuoteParams fields:
     - `leasePeriodMonths`: z.number() with refine to match LEASE_PERIOD_OPTIONS
     - `residualMethod`: z.enum(['VEHICLE_PRICE', 'ACQUISITION_COST'])
     - `residualRate`: z.number().min(0).max(1)
     - `depositRate`: z.number().min(0).max(0.4)
     - `advancePayment`: z.number().min(0)
     - `creditGroup`: z.union([z.literal(1), z.literal(2), z.literal(3)])

3. Create `src/features/inventory/actions/generate-quote.ts`:
   - `'use server'` directive
   - Export `generateQuote(vehicles: InventoryVehicleForQuote[], params: QuoteParams): Promise<QuoteGenerationResult>`
   - Validate params with quoteParamsSchema. On failure, throw Error with Zod issues.
   - For each vehicle:
     a. Calculate effectivePrice: vehiclePrice - (subsidyAmount ?? 0) - (vehiclePrice * (promotionRate ?? 0))
     b. Build QuoteInput using effectivePrice as vehiclePrice, vehicle's brand/category/fuelType/isImport, and user's params
     c. Call `calculateQuote(input)` to get lease result
     d. Call `estimateMonthlyRental(effectivePrice, params.leasePeriodMonths)` for rental comparison (import from @/lib/finance)
   - Return `{ params, vehicles: results, generatedAt: new Date() }`
