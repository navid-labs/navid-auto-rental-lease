1. Add three new Prisma models to schema.prisma (after ResidualValueRate):

```prisma
model PromoRate {
  id        String   @id @default(uuid()) @db.Uuid
  brandId   String   @map("brand_id") @db.Uuid
  rate      Decimal  // e.g. 0.03 = 3% promo discount
  label     String?  // e.g. "봄 프로모션"
  isActive  Boolean  @default(true) @map("is_active")
  createdAt DateTime @default(now()) @map("created_at")
  updatedAt DateTime @updatedAt @map("updated_at")

  brand Brand @relation(fields: [brandId], references: [id])

  @@unique([brandId])
  @@map("promo_rates")
}

model Subsidy {
  id        String   @id @default(uuid()) @db.Uuid
  brandId   String?  @map("brand_id") @db.Uuid  // null = global subsidy
  name      String   // e.g. "전략구매 보조금"
  amount    Int      // KRW amount
  isActive  Boolean  @default(true) @map("is_active")
  createdAt DateTime @default(now()) @map("created_at")
  updatedAt DateTime @updatedAt @map("updated_at")

  brand Brand? @relation(fields: [brandId], references: [id])

  @@map("subsidies")
}

model DefaultSetting {
  id        String   @id @default(uuid()) @db.Uuid
  key       String   @unique // "annual_rate", "residual_rate", "settings_password"
  value     String   // stored as string, parsed by consumer
  label     String   // display name
  updatedAt DateTime @updatedAt @map("updated_at")

  @@map("default_settings")
}
```

Add `promoRates PromoRate[]` and `subsidies Subsidy[]` relation arrays to the Brand model.

5. Create `src/features/settings/actions/settings.ts`:
Follow the exact pattern from `residual-rate.ts` (ActionResult type, getCurrentUser admin check, Zod validation, revalidatePath).
- `getPromoRates()`: List all promo rates with brand name, ordered by brand name
- `upsertPromoRate(data)`: Create or update promo rate by brandId unique constraint
- `deletePromoRate(id)`: Delete promo rate
- `getSubsidies()`: List all subsidies with optional brand name
- `upsertSubsidy(data)`: Create or update subsidy (use id for update, omit for create)
- `deleteSubsidy(id)`: Delete subsidy
- `getDefaultSettings()`: List all default settings
- `upsertDefaultSetting(data)`: Update a default setting by key (upsert)

All mutations: check admin role, validate with Zod, revalidatePath('/admin/settings').
