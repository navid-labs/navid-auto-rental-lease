```prisma
model InventoryItem {
  id              String   @id @default(uuid()) @db.Uuid
  category        String   // 구분: 전략구매/일반구매
  itemNumber      String   @map("item_number") // 번호
  promo           String?  // 프로모션
  representModel  String   @map("represent_model") // 대표차종
  modelName       String   @map("model_name") // 차종명
  options         String?  // 옵션
  modelYear       Int      @map("model_year") // 차량연식
  exteriorColor   String?  @map("exterior_color") // 외장색
  interiorColor   String?  @map("interior_color") // 내장색
  price           Int      // 가격 (KRW)
  subsidy         Int      @default(0) // 보조금
  availableQty    Int      @default(0) @map("available_qty") // 판매가능수량
  readyQty        Int      @default(0) @map("ready_qty") // 즉시출고수량
  productionDate  String?  @map("production_date") // 생산예시일
  notice          String?  // 공지
  createdAt       DateTime @default(now()) @map("created_at")
  updatedAt       DateTime @updatedAt @map("updated_at")

  @@map("inventory_items")
}
```

2. Create `src/features/inventory/schemas/inventory-upload.ts`:
- `inventoryRowSchema`: Zod schema matching CSV row structure
  - category: z.string().min(1)
  - itemNumber: z.string().min(1)
  - promo: z.string().optional()
  - representModel: z.string().min(1)
  - modelName: z.string().min(1)
  - options: z.string().optional()
  - modelYear: z.coerce.number().int().min(2000).max(2030)
  - exteriorColor, interiorColor: z.string().optional()
  - price: z.coerce.number().int().min(0)
  - subsidy: z.coerce.number().int().min(0).default(0)
  - availableQty: z.coerce.number().int().min(0).default(0)
  - readyQty: z.coerce.number().int().min(0).default(0)
  - productionDate: z.string().optional()
  - notice: z.string().optional()
- Export `type InventoryRowInput = z.infer<typeof inventoryRowSchema>`

3. Create `src/features/inventory/actions/inventory-upload.ts`:

`uploadInventoryCsv(formData: FormData)`:
- Admin role check via getCurrentUser()
- Extract file from FormData, read as text
- Parse CSV: split by newlines, first line = headers, rest = data rows
- Map Korean headers to schema fields: {"구분":"category", "번호":"itemNumber", "프로모션":"promo", "대표차종":"representModel", "차종명":"modelName", "옵션":"options", "차량연식":"modelYear", "외장색":"exteriorColor", "내장색":"interiorColor", "가격":"price", "보조금":"subsidy", "판매가능수량":"availableQty", "즉시출고수량":"readyQty", "생산예시일":"productionDate", "공지":"notice"}
- Validate each row with inventoryRowSchema.safeParse()
- Collect valid rows and error rows (with row number + error message)
- If valid rows exist: delete all existing InventoryItems (full refresh strategy), then createMany with valid data
- Update or create a DefaultSetting record with key="last_inventory_upload" and value=new Date().toISOString()
- revalidatePath('/admin/inventory')
- Return `{ success: true, imported: validCount, errors: errorRows[] }` or `{ error: string }`

`getLastUploadTime()`:
- Query DefaultSetting where key="last_inventory_upload"
- Return the value (ISO string) or null
