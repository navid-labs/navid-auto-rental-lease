    **1. Create `body-diagram-svg.tsx`** -- SVG paths for sedan silhouette (React.memo wrapped):

    This component renders a simplified geometric sedan outline with named panel paths. NOT photorealistic -- clean, schematic line drawings similar to insurance claim diagrams.

    ```typescript
    'use client'
    import { memo } from 'react'

    type PanelStatus = 'normal' | 'repainted' | 'replaced'
    type BodyDiagramSvgProps = {
      view: 'front' | 'rear' | 'left' | 'right' | 'top'
      panels: Record<string, PanelStatus>
      hoveredPanel: string | null
      onPanelHover: (panelKey: string | null) => void
      onPanelClick: (panelKey: string) => void
    }
    ```

    **SVG structure for each view** (use `viewBox="0 0 400 300"` with `width="100%" preserveAspectRatio="xMidYMid meet"`):

    - **Top view**: Shows hood, roof, trunk, left/right fenders, left/right doors, left/right rockers as distinct `<path>` elements. Sedan silhouette from above.
    - **Front view**: Shows hood, frontBumper, frontLeftFender, frontRightFender as distinct panels.
    - **Rear view**: Shows trunk, rearBumper, rearLeftFender, rearRightFender.
    - **Left view**: Shows frontLeftFender, frontLeftDoor, rearLeftDoor, rearLeftFender, leftRocker.
    - **Right view**: Mirror of left view with right panel keys.

    Each `<path>` element:
    - `data-panel={panelKey}` attribute
    - `fill` from PANEL_COLORS based on panel status
    - `stroke="#94A3B8"` (slate-400) with `strokeWidth="1.5"`
    - `className="cursor-pointer transition-opacity duration-200"`
    - `opacity={hoveredPanel === panelKey ? 1 : 0.8}`
    - `onMouseEnter={() => onPanelHover(panelKey)}`
    - `onMouseLeave={() => onPanelHover(null)}`
    - `onClick={() => onPanelClick(panelKey)}`

    SVG paths should be simplified geometric shapes (rectangles with rounded corners, trapezoids for fenders). Each panel must be visually distinct and clickable. Use simple path data -- this is a schematic, not an illustration.

    Wrap the component in `memo()` to prevent expensive SVG re-renders.
