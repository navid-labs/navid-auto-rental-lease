## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding |
| sm | 8px | Compact element spacing, thumbnail gaps |
| md | 16px | Default section internal padding, card padding |
| lg | 24px | Section vertical spacing, sidebar padding |
| xl | 32px | Between major sections |
| 2xl | 48px | Page-level vertical rhythm between section groups |
| 3xl | 64px | Top/bottom page margins |

Exceptions:
- Touch targets: minimum 44px height for all tappable elements on mobile (CTA buttons, tab items, thumbnail images)
- Gallery main image container: aspect-ratio 4:3 (no fixed height)
- Sticky tab bar height: 48px
- Mobile bottom CTA bar height: 64px (matches existing `floating-cta.tsx`)

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px | 400 (Regular) | 1.6 | Spec values, descriptions, FAQ answers, review text |
| Label | 13px | 500 (Medium) | 1.4 | Section sub-labels, category names, badge text, metadata |
| Heading | 18px | 600 (Semibold) | 1.3 | Section titles (가격정보, 기본정보, 옵션 etc.) |
| Display | 24px | 700 (Bold) | 1.2 | Vehicle name in header, price display (만원 단위) |

### Component-Scoped Overrides
These deviate from the 4-role scale for specific component contexts:

| Context | Size | Weight | Rationale |
|---------|------|--------|-----------|
| Sidebar price | 20px | 700 (Bold) | Sidebar is narrower than main content; Display 24px is too large |
| Grade badge | 16px | 700 (Bold) | Badge content needs emphasis but Heading 18px is too large for inline badge |

Additional type rules:
- Price numbers: `tabular-nums` font-feature for alignment (Pretendard supports this)
- 만원 unit suffix: 14px Regular, same line as price Display

## Interaction Patterns

### Gallery (DETAIL-01)
- Main image: click/tap opens YARL lightbox at current index
- Thumbnail: click scrolls main carousel to that image
- Category tab: filters images, resets carousel to index 0 (use `key={activeCategory}` for Embla remount)
- Image count badge: "📷 {count}" overlay on main image bottom-right, `bg-black/60 text-white text-xs px-2 py-1 rounded`
- Mobile: swipe horizontal on main image, dot indicators instead of thumbnail strip

### Body Diagram (DETAIL-04)
- Desktop: hover panel → tooltip showing panel name + status (정상/판금/교환)
- Mobile: tap panel → same tooltip (dismisses on tap elsewhere)
- SVG panels use `cursor-pointer` and `transition-opacity duration-200`
- Active (hovered/tapped) panel: `opacity-100`, inactive: `opacity-80`

### Scroll-Spy Navigation
- Sticky below header (top-16 on desktop accounting for header height)
- Active tab: accent underline (`border-b-2 border-accent`)
- Click tab: smooth scroll to section with offset for header + tab bar height
- Horizontal scroll on mobile with `overflow-x-auto scrollbar-hide`

### Sticky Sidebar (DETAIL-10)
- Sticky within 7:3 flex container, `top-32` (header 64px + tab bar 48px + gap 16px)
- "구매하기" button: `bg-accent text-white w-full h-12 text-base font-semibold rounded-lg`
- Other CTAs: `variant="outline"` full width
- Icon buttons (찜/비교/공유): horizontal row, `variant="ghost"` with icons

### Expandable Sections
- Options "옵션 모두 보기": Collapsible, shows first 8 icons by default, expand reveals all
- Diagnosis "전체보기": Collapsible, shows summary cards by default, expand reveals category details
- Trigger: `text-accent text-sm font-medium cursor-pointer` with chevron icon

## Loading States

### Skeleton Design
- Gallery: `aspect-[4/3] bg-muted animate-pulse rounded-lg` + 5 small thumbnail rectangles below
- Price section: 2 lines of varying width `h-6 bg-muted rounded`
- Basic info: 2x3 grid of `h-4 bg-muted rounded` cells
- Options: 3x4 grid of `h-10 w-10 bg-muted rounded-full` (icon placeholders)
- Body diagram: single `aspect-[2/1] bg-muted rounded-lg` rectangle
- Sidebar: `h-[400px] bg-muted rounded-lg`
- Each section skeleton wrapped in `<Card className="p-6">` for consistency

### Error States
- Page-level error: centered card with `AlertTriangle` icon + error copy + "다시 시도" button
- Image load error: `bg-muted` placeholder with `ImageOff` icon centered
- Section data missing (null JSONB): show empty state (not error state) — data absence is expected for some vehicles
