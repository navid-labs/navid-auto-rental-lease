    **1. Create `src/lib/validation/image.ts`:**

    ```typescript
    const MAX_FILE_SIZE = 10 * 1024 * 1024 // 10MB

    const ALLOWED_MIME_TYPES = new Set([
      'image/jpeg',
      'image/png',
      'image/webp',
      'image/gif',
    ])

    // Magic byte signatures for image formats
    const IMAGE_SIGNATURES = [
      { mime: 'image/jpeg', bytes: [0xFF, 0xD8, 0xFF] },
      { mime: 'image/png', bytes: [0x89, 0x50, 0x4E, 0x47] },
      { mime: 'image/gif', bytes: [0x47, 0x49, 0x46, 0x38] },
      // WebP: starts with RIFF....WEBP
      { mime: 'image/webp', bytes: [0x52, 0x49, 0x46, 0x46], offsetBytes: { offset: 8, bytes: [0x57, 0x45, 0x42, 0x50] } },
    ]
    ```

    Export `validateImageFile(file: File): Promise<{ valid: true } | { error: string }>`:
    - Check file is not null
    - Check `file.size <= MAX_FILE_SIZE`, error: `'파일 크기는 10MB 이하만 허용됩니다.'`
    - Check `ALLOWED_MIME_TYPES.has(file.type)`, error: `'허용되지 않는 파일 형식입니다. JPEG, PNG, WebP, GIF만 업로드 가능합니다.'`
    - Read first 12 bytes via `new Uint8Array(await file.slice(0, 12).arrayBuffer())`
    - Check magic bytes match at least one entry in `IMAGE_SIGNATURES`:
      - For each signature, compare `header[i] === sig.bytes[i]` for all bytes
      - For WebP, additionally check `offsetBytes`: compare bytes at specified offset
    - If no signature matches, error: `'파일 내용이 이미지 형식과 일치하지 않습니다.'`
    - Return `{ valid: true }` if all checks pass

    Also export `MAX_FILE_SIZE` for reuse.
