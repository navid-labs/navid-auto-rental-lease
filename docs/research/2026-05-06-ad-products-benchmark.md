## 한 줄 결론

엔카는 **18종 SKU의 풍부한 상품 카탈로그** (개당 27,000원 ~ 패키지 192,000원, 자유이용권 12개월 53,280,000원까지)로 매도자 광고를 다층화. 이어카는 **단일 빠른승계 패키지(시세 4%)**로 단순화. KB는 **직거래 광고 무료** + Pro 경매 모델. 헤이딜러는 **딜러 매수(B2B) 모델**로 매도자 광고 자체가 없음. **차용 권장 = 이어카형 단일 SKU에서 시작 → 엔카형 다층 SKU로 확장 (Phase 3)**.

## 3. KB차차차 내차팔기 (3종 분기)

### 3.1 Pro (대행 모델)

```
01 차량번호 입력 → 02 평가사 방문 예약 → 03 카옥션 경매 진행
```

- 누적 신청 481,575건
- 카옥션(B2B 경매) 통합 → 딜러 입찰
- 매도자 부담 수수료: 미공개 (페이지 노출 X)

### 3.2 Self (셀프 경매)

```
01 차량번호 입력 → 02 추가정보(사진) → 03 48시간 경매 → 최고가 딜러 거래
```

- 48시간 경매 후 자동 종료
- 매도자 부담 수수료: 미공개

### 3.3 직거래

```
01 차량광고 직접등록 (서비스 이용수수료 무료) → 02 구매문의 접수 → 03 판매완료 처리
```

- **서비스 이용수수료 무료** ⭐
- 누적 74,235건 (Pro 대비 1/6 규모)
- 차용 모델과 가장 유사 (광고 + 채팅)

→ KB는 **무료 직거래 + 유료 대행/경매**의 이중 모델. 차용은 KB 직거래와 본질적 유사.
