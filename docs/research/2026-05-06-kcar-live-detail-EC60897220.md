## 차용 미러링 우선순위 (재정렬)

K카 라이브 데이터 분석 후 우선순위 갱신:

### Tier 1 (Phase 1 — 즉시)

1. **다중 이미지 갤러리 (15~30장 권고)** — 사진 갯수가 신뢰의 1차 지표. 차용은 placeholder 단일 → mockup data부터 8~11장 보강 (TASK-005에 이미 반영됨).
2. **카테고리 옵션 표시** — TASK-006으로 헬퍼 만들고 detail 페이지 섹션 추가.
3. **계약 정보 카드** — TASK-008.
4. **신뢰 stat 4개 카드** — 헤더 직하 (기본정보/주행거리/내차피해/진단결과). VehicleCard에서 재사용.

### Tier 2 (Phase 1.5 — schema 확장 후)

5. **진단 결과 표** — 외판/주요부 판금·교환 카운트 + 타이어 잔존량 + 소모품 통과 (`inspectionChecklist: Json` 표준화).
6. **보험이력 박스** — 6개 항목 + 특수사고. 신규 필드 또는 Json 키.
7. **평가사 코멘트 4섹션 카드** — DEALER role의 review 텍스트. Profile 확장 또는 별도 모델.
8. **선택옵션 출고가 테이블** — 옵션을 `{ name, price }[]` Json으로 확장.

### Tier 3 (Phase 2)

9. **해시태그** — `#무사고 #1인소유 #짧은주행` 자동 추출 룰 (사고/소유주 횟수/주행거리 기반).
10. **차체 부위 SVG 다이어그램** — 시각화 라이브러리 또는 SVG sprite (sedan/suv/van 별).
11. **상세 페이지 내 후기 + FAQ 임베드** — 매물 단위 X, 카테고리 단위.
12. **3사 결제 방식 미러링** — 토스 단일에서 → 가상계좌/카드/할부 분기.
