## 1. 엔카 패키지광고 슬롯 상세 (`sell.encar.com/ad-package/preview`)

### 1.1 슬롯 어휘 13종 (전수)

| 분류 | 슬롯 |
|---|---|
| **패키지 묶음** | 베이직, 프로, 슈퍼 |
| **모바일 위치** | 모바일 프리미엄, 모바일우대, 모바일 강조형, 모바일 기본형 |
| **PC 위치** | 우대등록, 사진우대 |
| **자동 갱신** | 자동업데이트 라이트, 자동업데이트 스탠다드, 자동업데이트 맥스 |
| **강조 배지** | 핫마크 |

### 1.2 베이직 패키지 효과 명세 (대당 45,000원, 정가 48,000원)

```
구성상품: 자동업데이트 라이트 + 모바일 강조형

PC 노출:
  - 사진우대 + 우대등록 + 기본형 영역 노출
  - 리스트에서 핫마크 + 한줄광고 노출

모바일 노출:
  - 기본형 영역 노출
  - 상세페이지에서 핫마크 노출
```

→ **베이직만으로도 5개 슬롯 효과**. 프로/슈퍼는 더 강한 노출 + 모바일 우대/프리미엄 추가.

### 1.3 차용 PromotionSlot enum 보강 (v2 5종 → v3 9종)

```prisma
enum PromotionSlot {
  // v1/v2 기존
  LIST_TOP                  // 엔카 '우대등록' 매핑
  HOME_CAROUSEL             // 엔카 '모바일 프리미엄'
  CATEGORY_BADGE            // 엔카 '핫마크'
  PHOTO_PRIORITY            // 엔카 '사진우대'
  AUTO_REFRESH              // 엔카 '자동업데이트 라이트'
  AI_PRESS                  // 이어카 '/premium/<id>'
  QUICK_PACKAGE             // 빠른승계 통합
  AGENT_BADGE               // 이어카 'badge_quick.png'

  // v3 신규 (엔카 슬롯 어휘 보강)
  MOBILE_FEATURED           // 모바일 강조형 (베이직 핵심)
  MOBILE_PRIORITY           // 모바일우대 (프로/슈퍼)
  ONELINE_AD                // 한줄광고 (리스트 카드 위 광고문)
  AUTO_REFRESH_PLUS         // 자동업데이트 스탠다드/맥스
}
```

→ **엔카는 PC ↔ 모바일 슬롯을 분리 과금**. 차용은 모바일 우선 디자인이라 단순화 가능 (`LIST_TOP` 하나로 양쪽 커버).

## 2. KB 직거래/Pro 수수료 — 무료 확정

WebSearch + 페이지 본문 종합:

| KB 모델 | 수수료 |
|---|---|
| **Pro** (대행 → 카옥션) | KB 자체 수수료 **미공개**. 딜러 마진(시세 대비 감가)으로 흡수 추정 |
| **Self** (셀프 48시간 경매) | 동일 — 마진형 |
| **직거래** | **서비스 이용수수료 무료** (KB 명시) |
| 등록신청대행 (구매자) | 단지별 1~2.5만원 (구매자 부담) |

> *"다른 곳보다 저렴한 가격에 구입했고 판매 차량도 기존 상담가격과 다름없이 받았다"* — KB 후기
> *"KB차차차가 초기 견적은 높으나 후기와 감가 때문에 최종 판매가가 떨어질 수 있다"* — clien.net 후기

→ **차용 (d) 결정과 정합**: KB 직거래처럼 등록 무료. 차용은 추가로 빠른승계 4% 수수료 옵션.

## 3. 헤이딜러 — 딜러 부담 수수료 (정확값)

### 3.1 매도자 부담

**0원** (헤이딜러는 판매자에게 수수료 받지 않음).

### 3.2 딜러 부담 — 셀프 등록

| 차량가 | 딜러 수수료 |
|---|---|
| 500만원 ~ 3,000만원 | 235,000원 |
| 3,001만원 이상 | 345,000원 |

### 3.3 딜러 부담 — 헤이딜러 제로

| 차량가 | 딜러 수수료 |
|---|---|
| ~ 1,000만원 | 290,000원 |
| 1,001만원 ~ 1,500만원 | 350,000원 |
| 1,501만원 ~ 3,000만원 | 380,000원 |
| 3,001만원 ~ 4,000만원 | 460,000원 |
| 4,001만원 이상 | 490,000원 |

### 3.4 운영 모델

- 차량 사진/정보 등록 → 200+ 딜러 비공개 경매 입찰 → 매도자 최고가 선택
- B2C(매도자) ↔ B2B(딜러) 양면 시장
- 차용에 직접 적용 X (차용은 C2C 거래 모델). 단 **미래 B2B 입찰 옵션 도입 시** 헤이딜러 가격대를 reference.

## 7. 미해결 (Phase 3+ 별도 조사 필요)

- **엔카 프로/슈퍼 패키지 효과 명세** — preview 페이지에 모달로 들어가 있어 firecrawl 단일 scrape로 못 잡힘. interact 모드 또는 Playwright 필요.
- **KB Pro 카옥션 딜러 수수료** — 카옥션 딜러 가입 후 노출 추정. 별도 조사.

- **헤이딜러 제로 vs 셀프 차이** — "제로"는 헤이딜러가 직접 매수해주는 것 (위험 부담 X), "셀프"는 딜러 입찰 (위험 부담 ↑). 정확한 차이는 추가 fetch 필요.

## 8. 결론 — 차용 의사결정에 미치는 영향

1. **PromotionSlot enum 8종 (v2) → v3는 4종 추가하지만 MVP는 v2 유지** — 단순함 유지.
2. **MVP 가격 인하 권장**: Phase 2 SKU 가격을 v1 대비 50~60% 하향 (헤이딜러 reference + 차용 트래픽 미축적 고려).
3. **KB 직거래 무료 모델과 차용 (d) 정합 확인** — 등록 무료 + 빠른승계 옵션은 정상.
4. **헤이딜러 가격대 ≠ 차용 reference** — 헤이딜러는 딜러 부담, 차용은 매도자 부담. 직접 비교 부적합. 단 시장 가격 감각으로만 활용.
5. **엔카 디스플레이 광고는 후순위** — Phase 5+, 자체 트래픽 축적 후.
